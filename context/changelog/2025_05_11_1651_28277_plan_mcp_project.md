## Implementation Details

### Server Structure

The MCP server should follow this basic structure:

```typescript
// Basic server class
export class McpServer {
  private tools: Record<string, McpTool> = {};
  private systemPrompt: string;

  constructor({ systemPrompt }: { systemPrompt: string }) {
    this.systemPrompt = systemPrompt;
  }

  registerTool(tool: McpTool): void {
    this.tools[tool.name] = tool;
  }

  async handleRequest(request: McpRequest): Promise<McpResponse> {
    // Handle request logic
  }
}
```

### Tool Interface

Tools should implement this interface:

```typescript
interface McpTool {
  name: string;
  description: string;
  parameters: Record<string, McpParameter>;
  handler: (params: Record<string, any>) => Promise<any>;
}

interface McpParameter {
  type: string;
  description: string;
  required: boolean;
  default?: any;
}
```

### Greeting Tool Implementation

```typescript
const greetingTool: McpTool = {
  name: "greeting",
  description: "A tool that creates a personalized greeting",
  parameters: {
    salutation: {
      type: "string",
      description: "The greeting to use",
      required: false,
      default: "Hello"
    },
    name: {
      type: "string",
      description: "The name to greet",
      required: false,
      default: "World"
    }
  },
  handler: async ({ salutation = "Hello", name = "World" }) => {
    return `${salutation}, ${name}!`;
  }
};
```

## Testing Approach

Each component should have unit tests covering:

1. Base Cases - Type verification, simplest calls work
2. Error Conditions - Parameter validation, error handling
3. Happy Paths - Common use cases with expected parameters
4. Specific Scenarios - Edge cases like empty strings, special characters

Example test for greeting tool:

```typescript
describe("Greeting Tool", () => {
  // Base Cases
  describe("Base Cases", () => {
    it("is an object with the expected shape", () => {
      expect(greetingTool).toHaveProperty("name", "greeting");
      expect(greetingTool).toHaveProperty("description");
      expect(greetingTool).toHaveProperty("parameters");
      expect(greetingTool).toHaveProperty("handler");
      expect(typeof greetingTool.handler).toBe("function");
    });
    
    it("returns a string", async () => {
      const result = await greetingTool.handler({});
      expect(typeof result).toBe("string");
    });
  });
  
  // Happy Paths
  describe("Happy Paths", () => {
    it("returns 'Hello, World!' with default parameters", async () => {
      const result = await greetingTool.handler({});
      expect(result).toBe("Hello, World!");
    });
    
    it("returns custom greeting with provided parameters", async () => {
      const result = await greetingTool.handler({ 
        salutation: "Ahoy", 
        name: "Captain" 
      });
      expect(result).toBe("Ahoy, Captain!");
    });
  });
});
```
