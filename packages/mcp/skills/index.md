# Jaypie Skills

Welcome to Jaypie development documentation. These skills provide guidance for building cloud applications with the Jaypie framework.

## Getting Started

1. **New to Jaypie?** Start with `skill("introduction")` for an overview
2. **Building infrastructure?** Check `skill("cdk")` for CDK patterns
3. **Writing tests?** See `skill("tests")` and `skill("mocks")`
4. **Debugging issues?** Use `skill("debugging")` and `skill("logs")`
