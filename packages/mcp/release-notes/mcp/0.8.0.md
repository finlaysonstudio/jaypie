## Breaking Changes

- Removed the `aws` MCP tool (Lambda, S3, SQS, CloudWatch, Step Functions, CloudFormation, DynamoDB commands)
- Removed the `llm` MCP tool (debug_call, validate commands)
- Removed `mcp-aws`, `mcp-dynamodb`, and `mcp-llm` skill files
- Removed `@jaypie/llm` dependency

## Changes

- Skill index now returns a flat `# Index of Skills` listing (removed `index.md` dependency)

## Notes

- Skill documentation for AWS, DynamoDB, and LLM remains available (`skill("aws")`, `skill("dynamodb")`, `skill("llm")`)
- MCP server now provides 4 tools: `skill`, `version`, `release_notes`, `datadog`
