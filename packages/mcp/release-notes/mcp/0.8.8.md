## Changes

- Datadog MCP tool no longer auto-appends `DD_SOURCE` or `DD_SERVICE` env vars to search queries; source and service filters must be passed explicitly as parameters
- Added CI/CD build-order guidance to subpackage, cicd, cdk, and lambda skills for new packages that need `dist/` built before CDK synth
