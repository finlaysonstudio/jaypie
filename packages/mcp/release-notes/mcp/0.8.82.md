## Changes

- Release-note frontmatter parsing now uses `parseFrontmatter` from
  `@jaypie/kit` instead of `gray-matter`. Removes the `gray-matter` dependency
  (and its local type shim), eliminating the transitive `js-yaml@3` advisory
  (GHSA-h67p-54hq-rp68) from the published package.
- Requires `@jaypie/kit@^1.2.11` (parseFrontmatter) and
  `@jaypie/tildeskill@^0.3.3`.

No runtime behavior changes.
