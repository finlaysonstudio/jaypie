## Changes

### New Features

- **skill tool**: New `skill(alias)` function for accessing Jaypie documentation
  - Call `skill("index")` to list all available skills
  - Call `skill("aws")`, `skill("tests")`, etc. to get specific documentation
  - Path traversal protection for security
  - Case insensitive alias matching

### New Skills

Added 21 skill files covering all aspects of Jaypie development:
- aws, cdk, cicd, datadog, debugging, dns, dynamodb, errors, fabric
- index, introduction, logs, mocks, models, secrets, services
- style, tests, tools, topics, variables

### Deprecations

- `list_prompts` - Use `skill("index")` instead
- `read_prompt` - Use `skill(alias)` instead

Both deprecated tools remain functional for backwards compatibility.
