## Changes

- Added `removalPolicy` prop to `JaypieEnvSecret`
  - Accepts `boolean` (`true` = RETAIN, `false` = DESTROY) or CDK `RemovalPolicy` enum
  - Simplifies production retention: `removalPolicy: isProductionEnv()`
