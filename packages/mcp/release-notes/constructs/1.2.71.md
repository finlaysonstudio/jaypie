## Changes

- `JaypieDynamoDb` enables DynamoDB TTL on the `ttl` attribute by default (matching `@jaypie/dynamodb`). Pass `timeToLiveAttribute: "<name>"` to use a different attribute, or `timeToLiveAttribute: false` to disable TTL. Enabling TTL on a table whose items never set `ttl` is inert and free.
