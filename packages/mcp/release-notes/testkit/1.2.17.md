# @jaypie/testkit 1.2.17

## Changes

Updated dynamodb mock exports to match `@jaypie/dynamodb` 0.4.0 vocabulary changes:

### Constants
- `INDEX_CLASS` → `INDEX_CATEGORY`

### Functions
- `buildIndexClass` → `buildIndexCategory`
- `queryByClass` → `queryByCategory`

## Migration

Update your test imports:

```typescript
// Before
import { INDEX_CLASS, buildIndexClass, queryByClass } from "@jaypie/testkit/mock";

// After
import { INDEX_CATEGORY, buildIndexCategory, queryByCategory } from "@jaypie/testkit/mock";
```
