## Changes

- The markdown store now reads and writes frontmatter via `parseFrontmatter`
  and `stringifyFrontmatter` from `@jaypie/kit` instead of `gray-matter`.
  Removes the `gray-matter` dependency, eliminating the transitive `js-yaml@3`
  advisory (GHSA-h67p-54hq-rp68) from the published package.
- Requires `@jaypie/kit@^1.2.11`.

No runtime behavior changes.
