## Changes

- `registerModel({ model, ttl })` accepts an optional default TTL on `ModelSchema` (`number | string`). Validated at registration; `getModelSchema(model).ttl` exposes it. `@jaypie/dynamodb`'s `resolveTtl` interprets the value at write time.
- Add `DEFAULT_TTL_ATTRIBUTE` (`"ttl"`) constant.
